# Použití

- Vytvořit složku "images" ve stejném souboru jako je poznávačka.py  
- Vložit obrázky do složky images
- Spustit program

Přiklad obsahu složky:
```
složka
├── images
│   ├── example1.jpg
│   ├── example2.jpg
│   └── example3.jpg
└── poznávačka.py
```
program uznává jako správnou odpověď text před příponou `.jpg`
